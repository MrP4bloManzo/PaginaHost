Categorias de Software

Categoría Qué es Ejemplos Subcategorías
Hardware El hardware se refiere a los componentes físicos de una computadora o dispositivo electrónico. Ejemplos: CPU, memoria RAM, disco duro, monitor, teclado.
  • Procesadores
  • Dispositivos de entrada
  • Dispositivos de salida
  • Almacenamiento
  • Tarjetas gráficas
Sistema Operativo El sistema operativo es un software que administra los recursos de hardware y proporciona servicios a los programas de aplicación. Ejemplos: Windows, macOS, Linux, Android, iOS.
  • Escritorio (para computadoras de escritorio y portátiles)
  • Móvil (para dispositivos móviles)
  • Servidores (para administrar servidores)
Software de Aplicación El software de aplicación son programas diseñados para realizar tareas específicas o resolver problemas. Ejemplos: Microsoft Word (procesador de texto), Adobe Photoshop (edición de imágenes), Google Chrome (navegador web).
  • Ofimática
  • Diseño gráfico
  • Navegadores web
  • Juegos
Software Malicioso El software malicioso (malware) es un tipo de software diseñado para dañar, robar información o realizar actividades perjudiciales en una computadora o dispositivo. Ejemplos: virus, troyanos, spyware, ransomware, gusanos.
  • Virus informáticos
  • Troyanos
  • Spyware
  • Ransomware
  • Gusanos
